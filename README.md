# OpenInvite Cards

This repo is associated with [openinvite.vip](https://openinvite.vip). The templates in this repo are hosted at [cards.openinvite.vip](https://cards.openinvite.vip) and can be used directly as templates in invite system.

Most of the templates here are listed in the template gallery at [openinvite.vip](https://openinvite.vip).

Contributions are welcome in this repo, but if you prefer, you may clone this repository and use the templates as bases for your own custom templates.

## New card script

Use `newcard.sh` to copy `base.html` into a new folder under `cards/`.

```bash
./newcard.sh christmas
```

This creates `cards/christmas/index.html` and exits with an error if the target already exists.
