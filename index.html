OpenInvite.vip

Invitation Templates, Ready for Injection

This repository holds standalone HTML templates used by the OpenInvite invitation system. Each template includes named divs (like title, date, time, and location) so the OpenInvite tool can inject event data into a beautiful design.

How it works

  1. Pick a template from the cards folder.
  2. OpenInvite injects data into empty named divs.
  3. Guests see a fully styled invitation card.

Template structure

Each card defines a full-page layout plus the OpenInvite RSVP scaffold styles. The injected fields include title, notes, date, time, location, address, map link, calendar link, and RSVP responses.

Create a new template with: ./newcard.sh theme-name

Theme coverage

The collection spans celebrations, holidays, and formal events with distinctive typography and color palettes.
